-------------------------------- MVP - Projeto Infotech --------------------------------
----------------------------------------------------------------------------------------
-------------------- Software com Arquitetura de Sistemas em Camadas -------------------

Instruções de instalação:
- Instalar um servidor MySQL (indico o XAMP)
- Criar a base de dados Infotec (Pasta 'SQL Banco', arquivo Infotec.sql)
- Executar o Instalador (setup.exe) 
- Cadastrar um produto para conseguir fazer uma venda

Funcionalidade do Software:
- Nova Venda (Atalho F2)
- Cria nova venda

- Cancela a Venda em Andamento (Atalho F3)
- Cancela a venda, se estiver iniciada

- Produtos (Atalho F4): 
- Adiciona os produtos na venda e cadastra novos produtos

- Cliente (Atalho F5)
- Só funciona com a venda iniciada
- Faz busca de cliente para adicionar na venda, cadastra novos clientes

- Consulta Vendas (Atalho F6)
- Só é ativado sem venda iniciada
- Consulta as vendas por data e/ou número

- Finalizar Venda (Atalho F10)
- Só funciona com a venda ativado
- Finaliza a venda e inclui a data de entrega

- Sair (Atalho F12)
- Finaliza o software

- Testes
- Executa os testes nos microserviçoes do programa
